# 自我进化插件

> 赋予你的 AstrBot **长期记忆**与**主动社交**能力，让机器人真正"认识"用户

> **⚠️ Wiki 版本说明**：本文档基于当前稳定版编写，Wiki 更新可能跟不上开发进度。如有疑问，优先 [提 Issue](https://github.com/Renyus/astrbot_plugin_self_evolution/issues) 或 [加群](#交流反馈) 询问。

---

## 核心亮点

| 功能 | 说明 |
|------|------|
| 🧠 **长期记忆** | 自动记住用户喜好、群聊约定，就像认识老朋友 |
| 💬 **主动参与** | 不只是等待@，会根据氛围智能插话 |
| 👤 **用户画像** | 为每个用户建立档案，对话更贴心 |
| ⚡ **精力系统** | 模拟真实疲劳感，让机器人更生动 |
| 💝 **情感积分** | 根据互动自动增减好感度 |

---

## 3分钟快速开始

```
Step 1: 安装插件 → AstrBot后台搜索"self_evolution"
Step 2: 创建知识库 → 取名"self_evolution_memory"
Step 3: 重载插件 → 完成！
```
